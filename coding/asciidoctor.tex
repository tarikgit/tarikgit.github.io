Simple asciidoctor files are compiled as follows:


\begin{verbatim}
asciidoctor sourcefile.adoc
\end{verbatim}

If the filecontains graphics, such as ditaa, use:


\begin{verbatim}
asciidoctor -r asciidoctor-diagram sourcefile.adoc
\end{verbatim}
